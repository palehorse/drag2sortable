
## drag2sortable
拖曳排序
## 前置安裝
    jQuery
## 安裝
1.用npm指令安裝
```sh
npm install drag2sortable
```
2.用html語法引入
```html
<script src="drag2sortable/drag2sortable.jquery.js"></script>
```
#### Vue
```javascript
require('drag2sortable/drag2sortable.jquery.js');
```
